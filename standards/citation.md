# Citation standard

Binding for `@cto-sources`, `@cto-update`, curricula references, and any recommendation of books/articles/talks.

## Required fields

Every cited work must include:

1. **Title**
2. **Author** and/or **URL** (canonical public page preferred)

Optional: year, format (book/article/talk/docs), Why (1 line).

## Honesty rules

- **Never invent** a book, paper, talk, or quote.
- If uncertain a source exists or matches the claim → mark **Unverified** or omit.
- Prefer primary/public sources (official docs, known books, reputable publishers).
- Do not fabricate page numbers, “studies show” statistics, or private URLs.

## When using web search

- Cite the concrete page found.
- If paywalled, still give title + author + publisher; note access constraint.

## Storage

Learner curation lives in `.training.cto/sources/`. Framework curricula may list well-known works by title + author; agents must not invent additional “classic” titles without verification.
