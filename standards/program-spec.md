# Program SPEC standard

Binding for `@cto-program-standard`, `@cto-program-custom`, `@cto-curriculum`.

## Required sections in PROGRAM.md

1. **Title + slug**
2. **Audience / level assumptions**
3. **Outcomes** (measurable, 3–6)
4. **Duration & cadence**
5. **Modules** — each with objectives, drill(s), sources, exit check
6. **Assessment / rubrics**
7. **Exit criteria** for program completion

## Design rules

- Prefer ≤6 modules unless multi-track.
- Every module needs a practice (drill or decision artifact).
- Sources follow citation.md.
- Adapt catalog copies to PROFILE (stage, constraints) without gutting outcomes.
- Custom programs: Confirm gate unless `-y`.

## Storage

Only under `.training.cto/programs/<slug>/`. Framework `curricula/` is read-only catalog.
