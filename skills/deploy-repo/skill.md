# deploy-repo

**Shell:** `bash <source>/scripts/deploy-repo.sh clone <url> <target>`  
**Archive:** `bash <source>/scripts/deploy-repo.sh archive <target-dir>`

Prefer `@deploy-basic` for multi-repo sharing; use this for standalone copies or backups.
