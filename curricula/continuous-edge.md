# Continuous Edge

**Slug:** `continuous-edge`  
**Duration:** Ongoing (monthly cadence)  
**Level:** Any — pairs with other programs

## Outcomes

- Living backlog of high-signal sources
- Monthly update ritual with practice implication
- Avoid trend addiction; keep durable principles

## Modules

### M1 — Source system
- Objectives: Install `@cto-sources` + `@cto-update` habit
- Drill: Curate 8 sources for your top gap area

### M2 — Monthly edge review
- Cadence: `@cto-update run` → pick 1 drill or mentor focus
- Artifact: `.training.cto/reports/update-YYYY-MM-DD.md`

### M3 — Teach-back
- Objectives: Explain one new idea as a 1-pager for your staff
- Exit: 3 consecutive monthly updates with at least one drill each quarter

## Assessment

Pass = ritual exists and changes behavior; fail = unread bookmark pile.

## Notes

Fed by `@cto-update`. Install alongside a primary program or alone for seasoned CTOs.
