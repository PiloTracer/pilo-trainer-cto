# Programs cheat sheet

| Want | Command |
|------|---------|
| List catalog | `@cto-program-standard list` |
| Install | `@cto-program-standard install - cto-foundations` |
| Custom | `@cto-program-custom - <request>` |
| Refine | `@cto-curriculum refine - <slug>` |
